## 基于 PowerCenter 的数据质量管控系统
[官网](http://www.dtinfor.cn)

用技术解决你的数据问题
### 我们做什么
我们通过智能校验、业务校验、监控PowerCenter流程状态等手段，对ETL运行过程中出现的问题进行及时的发现和统计
### 我们怎么做
 - 智能校验（记录预测，读取失败，写入失败）
 - 规则校验（定时触发，依赖触发，跨库合作）
 - 任务调度（定时调度，依赖调度，定时运行，失败自动尝试）
 - WEB化PowerCenter(启停工作流，流程运行状况，流程属性，节点详情)
